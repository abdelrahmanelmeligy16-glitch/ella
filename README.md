# E.L.L.A — Creator-Business Platform

**A full creator-business platform — editorial system, services, projects, store, community, and a narrated podcast — built solo and live in production.**

🔗 **Live:** https://ella-intelligence.com

---

## Highlights

- **47 routes · 98 TypeScript/TSX source files** — public editorial routes, detail pages, an admin dashboard, protected login, CRUD forms, and reusable UI primitives.
- **Supabase-backed** — migrations, typed DB interfaces, server actions, and row-level security; inquiry handling, comments, reactions, and subscribers.
- **Podcast system** — Season One with **30 narrated episodes** and a podcast RSS feed.
- **Production discipline** — fail-closed admin middleware, protected release gates, and CI alignment.

---

## Tech stack

`TypeScript` · `Next.js (App Router)` · `React` · `Node` · `Supabase / Postgres (RLS)` · `Tailwind` · `Vercel`
